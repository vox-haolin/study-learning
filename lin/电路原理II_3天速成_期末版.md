## 上午：实验1 电工仪表与误差分析

> 耗时：约1.5小时 | 目标：3分钟默写所有误差公式 + 3种误差秒区分

### 一、3分钟记住公式（必背）

#### 公式卡片（可直接截图背诵）

```
┌─────────────────────────────────────────────────┐
│             误差公式三兄弟                        │
│                                                 │
│  ① 绝对误差：△ = A_x − A₀                        │
│     （测量值 − 真实值，带正负号）                  │
│                                                 │
│  ② 相对误差：γ = (△/A₀) × 100%                   │
│     （绝对误差 ÷ 真实值，常用%）                   │
│                                                 │
│  ③ 近似相对误差：γ ≈ (△/A_x) × 100%               │
│     （当真实值未知时，用测量值代替）                 │
└─────────────────────────────────────────────────┘
```

#### 三种误差区分口诀 ★★★★★

```
系统误差：「有规律、总偏一边、可修正」
偶然误差：「没规律、忽大忽小、多次取平均」
疏忽误差：「读错了、记错了、人为犯的错」
```

| 误差类型 | 特征 | 处理方法 | 考试概率 |
|:---:|------|------|:---:|
| 系统误差 | 总是朝一个方向偏，有规律（如仪表零点漂移）| 校正、补偿 | ★★★★ |
| 偶然误差 | 时大时小无规律（如温度波动）| 多次测量取平均 | ★★★★ |
| 疏忽误差 | 读错、记错等人为明显错误 | 剔除异常值 | ★★★ |

#### 考试怎么问 → 你怎么答

| 考题问法 | 标准答案模板 |
|------|------|
| "电压表零点未校准，测出的数据属于什么误差？" | **系统误差**——有规律，总偏一边 |
| "读数时看错了刻度，属于什么误差？" | **疏忽误差**——人为明显错误 |
| "多次测量取平均值能消除什么误差？" | **偶然误差**——取平均可减小 |
| "量程10A的表测8A读数为8.1A，求绝对误差和相对误差" | △=8.1−8.0=**0.1A**，γ=0.1/8.0×100%=**1.25%** |

### 三、真题模拟 3题

**题1**（★★★★★）：用量程5V的电压表测量标准值为4.00V的电压，读数为4.15V。求绝对误差和相对误差。

> **套模板**：△=4.15−4.00=**0.15V**；γ=0.15/4.00×100%=**3.75%**

**题2**（★★★★）：判断以下场景的误差类型：
- (1) 万用表电池不足导致读数偏小 → **系统误差**
- (2) 实验室空调温度波动导致读数跳动 → **偶然误差**
- (3) 把2.35V读成2.53V → **疏忽误差**

**题3**（★★）：为什么相对误差比绝对误差更能评价测量准确度？

> **标准答案**：绝对误差只看偏差大小，不能反映测量本身的量级。比如测1V偏差0.1V（相对误差10%）和测100V偏差0.1V（相对误差0.1%）的准确度完全不同，只有相对误差才能公平比较。

KCL：「流入等于流出，节点电流总和为零」
KVL：「绕一圈回到原点，电压升降总和为零」

#### 考试怎么问 → 你怎么答

| 考题问法 | 标准答案模板 |
|------|------|
| "写出节点a的KCL方程" | I₁+I₂−I₃=0（流入为正）或 −I₁−I₂+I₃=0（流出为正），符号自己约定即可，但要统一 |
| "写出回路1的KVL方程" | 沿回路绕行方向：遇到电压升记+，电压降记−，加起来=0 |

### 二、叠加定理步骤口诀 ★★★★★

```
叠加定理四步走：
1. 画出每个独立源单独作用的分电路
2. 不作用的独立源要置零（电压源→短路，电流源→开路）
3. 分别计算各分电路的响应
4. 代数和叠加（注意正负号！）

⚠️ 受控源始终保留，千万别置零！
```

### 三、高频考点：功率不满足叠加 ★★★★★

> 这是考试中最爱挖坑的地方，必须背下来。

```
┌─────────────────────────────────────────────────┐
│  ⚠️ 为什么功率不满足叠加定理？                      │
│                                                 │
│  因为 P = I²R 是电流的二次函数，不是线性关系。       │
│  P(I₁+I₂) ≠ P(I₁) + P(I₂)                       │
│                                                 │
│  考试如果问："以下哪个量满足叠加定理？"              │
│  答案一定是电压U和电流I，绝不是功率P！              │
└─────────────────────────────────────────────────┘
```

#### 考试怎么问 → 你怎么答

| 考题问法 | 标准答案模板 |
|------|------|
| "功率是否满足叠加定理？为什么？" | **不满足**。P=I²R是二次关系，P(I₁+I₂)≠P(I₁)+P(I₂)，只有线性量（电压、电流）才满足 |
| "叠加定理适用于什么类型的电路？" | **线性电路**。非线性元件（二极管等）不适用 |
| "用叠加定理求电压U" | 画两个分电路→分别算U'和U"→U=U'+U" |

### 四、真题模拟 3题

**题1**（★★★★★）：如图，已知 I₁=2A, I₂=3A，方向均流入节点a，I₃流出节点a。求I₃。

> **套模板**：KCL → I₁+I₂−I₃=0 → I₃=2+3=**5A**

**题2**（★★★★★）：含一个12V电压源和一个2A电流源的电路，单独电压源作用时U'=4V，单独电流源作用时U"=6V，求总电压U。

> **套模板**：U=U'+U"=4+6=**10V**

**题3**（★★★★）：图中用叠加定理算出I'=1.5A，I"=0.5A，电阻R=2Ω。问电阻消耗功率是否等于两个分电路功率之和？

> **标准答案**：不等！P=(1.5+0.5)²×2=8W，P₁+P₂=1.5²×2+0.5²×2=4.5+0.5=5W，8W≠5W，功率不满足叠加定理。

## 晚上：实验3 戴维南定理 + 诺顿定理 + 最大功率传输

> 耗时：约2小时 | 目标：三步法求戴维南等效 + 最大功率传输公式倒背如流

### 一、戴维南/诺顿等效求法步骤 ★★★★★

#### 戴维南三步法口诀

```
戴维南三步走：
第一步：求Uoc（开路电压）—— 断开负载，求两端电压
第二步：求Req（等效电阻）—— 三种方法任选
第三步：画等效电路 —— Uoc串联Req，接上负载
```

#### 三种求Req的方法对比

| 方法 | 操作 | 适用场景 | 考试概率 |
|:---:|------|------|:---:|
| 开短法 | Req = Uoc / Isc | 内阻不太小（短路电流不过大） | ★★★★★ |
| 除源外加电源法 | 独立源置零→外加电压U→求电流I→Req=U/I | 含受控源时首选 | ★★★★ |
| 半偏法 | 调RL使U_RL=Uoc/2→此时Req=RL | 实验题最爱考 | ★★★ |

#### 诺顿定理速记

```
诺顿 = 戴维南的对偶
戴维南：电压源Uoc串联Req
诺  顿：电流源Isc并联Req

转换公式：Isc = Uoc / Req

口诀：「戴串诺并，乘除互换」
```

### 二、最大功率传输定理 ★★★★★

```
┌─────────────────────────────────────────────────┐
│         最大功率传输定理（必考！）                  │
│                                                 │
│  条件：RL = Req  （负载电阻 = 等效内阻）           │
│  最大功率：Pmax = Uoc² / (4·Req)                  │
│  此时效率：η = 50%                                │
│                                                 │
│  口诀：「匹配时功率最大，但效率只有一半」              │
└─────────────────────────────────────────────────┘
```

#### 考试怎么问 → 你怎么答

| 考题问法 | 标准答案模板 |
|------|------|
| "负载多大时获得最大功率？" | **RL = Req**（负载电阻等于等效内阻） |
| "最大功率是多少？" | Pmax = Uoc²/(4Req)，把Uoc和Req直接代入 |
| "为什么电力系统不用匹配？" | 匹配时**效率只有50%**，一半能量损耗在线路上，电力系统追求高效率而非最大功率 |
| "内阻很小时不能用什么方法测Req？" | **不能直接短路测Isc**（电流过大会烧设备），应用半偏法 |

### 三、套公式模板

#### 戴维南等效计算模板

```
已知电路，求对RL的戴维南等效。

步骤1（求Uoc）：
  断开RL，用KVL/KCL求开路电压
  Uoc = ______ V

步骤2（求Req）：
  方法选择：
  □ 开短法：Req = Uoc / Isc = ____ / ____ = ____ Ω
  □ 除源法：独立源置零 → 外加电压 → 求电流 → Req = ____ Ω
  □ 半偏法：调到U_RL = Uoc/2 → Req = RL = ____ Ω

步骤3（画等效）：
  ┌─Req─┬───┐
  │      │   │
  Uoc    RL  │
  │      │   │
  └──────┴───┘
```

#### 最大功率计算模板

```
已知 Uoc = ____ V，Req = ____ Ω

当 RL = Req = ____ Ω 时：
Pmax = Uoc²/(4×Req) = ____² / (4×____) = ____ W
效率 η = 50%
```

### 四、真题模拟 3题

**题1**（★★★★★）：某有源二端网络开路电压Uoc=12V，短路电流Isc=3A。求：(1) 等效电阻Req；(2) 负载RL=?时获最大功率，Pmax=?

> **套模板**：(1) Req = 12/3 = **4Ω**；(2) RL = Req = **4Ω**，Pmax = 12²/(4×4) = 144/16 = **9W**

**题2**（★★★★★）：测得有源二端网络开路电压10V，接上RL=5Ω时测得U_RL=5V（恰好为Uoc一半）。求Req和Pmax。

> **套模板**：U_RL=Uoc/2 → 半偏法 → **Req = RL = 5Ω**；Pmax = 10²/(4×5) = **5W**

**题3**（★★★）：内阻极小的有源网络（如新电池），能否用短路法测Req？为什么？应改用何法？

> **标准答案**：**不能**。内阻太小→Isc极大→可能烧毁电源或导线。应用**半偏法**或**除源外加电源法**。

## 上午：实验4 RC一阶电路暂态响应

> 耗时：约2小时 | 目标：三要素法5分钟出答案 + τ的物理意义秒答

### 一、三要素法步骤口诀 ★★★★★

```
三要素法解一阶电路：

要素①：初始值 f(0⁺)   —— 换路瞬间的值（利用换路定则）
要素②：稳态值 f(∞)    —— t→∞ 时的最终值
要素③：时间常数 τ      —— RC电路τ=RC，RL电路τ=L/R

公式：f(t) = f(∞) + [f(0⁺) − f(∞)] · e^(−t/τ)

口诀：「初值稳态先搞定，时间常数往里带」
```

#### 换路定则速记

```
┌──────────────────────────────────┐
│  换路瞬间（t=0⁺）：                │
│  u_C(0⁺) = u_C(0⁻)  电容电压不突变│
│  i_L(0⁺) = i_L(0⁻)  电感电流不突变│
│                                  │
│  口诀：「容压感流不突变」          │
└──────────────────────────────────┘
```

### 二、τ 的物理意义 ★★★★★

```
┌─────────────────────────────────────────────────┐
│  时间常数 τ 的物理意义（必考简答题！）              │
│                                                 │
│  τ = RC（RC电路）  或  τ = L/R（RL电路）          │
│                                                 │
│  充电：t=τ时，u_C = 63.2%·U_s（达到稳态的63.2%）   │
│  放电：t=τ时，u_C = 36.8%·U_0（剩下初始的36.8%）   │
│                                                 │
│  t=5τ 时：充放电完成约 99.3%，可认为进入稳态        │
│                                                 │
│  快速记忆法：                                     │
│  1τ → 63%                                        │
│  3τ → 95%                                        │
│  5τ → 99.3% ≈ 100%                               │
└─────────────────────────────────────────────────┘
```

#### 考试怎么问 → 你怎么答

| 考题问法 | 标准答案模板 |
|------|------|
| "时间常数的物理意义是什么？" | τ反映充放电快慢。τ越大充放电越慢。t=τ时充电到63.2%或放电到36.8%，t=5τ基本完成 |
| "RC电路中τ=?" | **τ=RC**（R单位Ω，C单位F，τ单位s） |
| "RL电路中τ=?" | **τ=L/R**（L单位H，R单位Ω，τ单位s）—— 注意不是R/L！ |
| "已知R=1kΩ, C=100μF, 求τ和5τ" | τ=1000×100×10⁻⁶=**0.1s**，5τ=**0.5s** |

### 三、微分电路 vs 积分电路 ★★★★

```
┌─────────────────────────────────────────────────┐
│  微分电路：τ << T（时间常数远小于信号周期）         │
│    → 从电阻两端取输出 → 输出尖脉冲（求导效果）      │
│                                                 │
│  积分电路：τ >> T（时间常数远大于信号周期）         │
│    → 从电容两端取输出 → 输出三角波（积分效果）      │
│                                                 │
│  口诀：「微阻积容，小快大慢」                       │
│  微分：电阻输出、τ小 → 快                      │
│  积分：电容输出、τ大 → 慢                      │
└─────────────────────────────────────────────────┘
```

### 四、套公式模板

#### 三要素法计算模板

```
电路参数：R=____Ω, C=____μF, U_s=____V

步骤1：求初始值
  换路前：u_C(0⁻) = ____ V → u_C(0⁺) = u_C(0⁻) = ____ V

步骤2：求稳态值
  t→∞ 时电容视为开路 → u_C(∞) = ____ V

步骤3：求时间常数
  τ = RC = ____ × ____ = ____ s

步骤4：代三要素公式
  u_C(t) = u_C(∞) + [u_C(0⁺) − u_C(∞)] · e^(−t/τ)
         = ____ + (____ − ____) · e^(−t/____)
         = ____ + ____ · e^(−t/____)  V
```

#### 充放电公式速查

```
充电（零状态响应）：u_C(t) = U_s(1 − e^(−t/τ))
放电（零输入响应）：u_C(t) = U_0 · e^(−t/τ)

全响应 = 零输入响应 + 零状态响应（满足叠加定理！）
```

### 五、真题模拟 3题

**题1**（★★★★★）：RC串联电路，R=2kΩ，C=50μF，U_s=10V，电容初始电压为0。(1)求τ；(2)求t=τ时u_C值。

> **套模板**：(1) τ=2000×50×10⁻⁶=**0.1s**；(2) u_C(τ)=10×(1−e⁻¹)=10×0.632=**6.32V**

**题2**（★★★★★）：已知零输入响应u_C(0⁺)=8V，τ=0.2s。t=0.2s时u_C=?

> **套模板**：u_C(0.2)=8×e^(−0.2/0.2)=8×e⁻¹=8×0.368=**2.944V**

**题3**（★★★）：方波信号周期T=1ms，R=1kΩ，C=0.1μF。判断此RC电路是微分还是积分？从何处取输出？

> **标准答案**：τ=RC=1000×0.1×10⁻⁶=**0.1ms**，τ<<T(1ms)→**微分电路**，从**电阻两端**取输出。

## 下午：实验5 交流电路参数测定

> 耗时：约2小时 | 目标：阻抗三角/功率三角背熟 + 三表法步骤秒默写

### 一、三个三角形速记 ★★★★★

```
┌─────────────────────────────────────────────────┐
│          交流电路三大三角形（必考！）               │
│                                                 │
│  阻抗三角形       电压三角形      功率三角形       │
│  Z = R + jX      U²=U_R²+U_X²    S²=P²+Q²       │
│                                                 │
│  |Z| = √(R²+X²)    U = √(U_R²+U_X²)  S = √(P²+Q²)│
│                                                 │
│  口诀：「阻抗R+X，电压有R有X，功率P加Q」             │
└─────────────────────────────────────────────────┘
```

#### 相位关系速记

```
感性负载：电压超前电流（φ>0）
容性负载：电流超前电压（φ<0）
纯电阻：  电压电流同相（φ=0）

感性负载等效：Z = R + jωL → |Z| = √(R²+(ωL)²)
容性负载等效：Z = R + 1/(jωC) = R − j/(ωC)
```

### 二、三表法测参数 ★★★★★

```
┌─────────────────────────────────────────────────┐
│       三表法测定交流阻抗参数（实验必考）            │
│                                                 │
│  电压表 → 测端口电压 U                            │
│  电流表 → 测回路电流 I                            │
│  功率表 → 测有功功率 P                            │
│                                                 │
│  然后算出：                                      │
│  |Z| = U / I          （阻抗模）                 │
│  cosφ = P / (U·I)    （功率因数）                 │
│  R = P / I² = |Z|·cosφ  （等效电阻）              │
│  X = √(|Z|² − R²)    （等效电抗）                 │
│                                                 │
│  口诀：「电压电流功率表，模阻感抗全搞定」              │
└─────────────────────────────────────────────────┘
```

#### 考试怎么问 → 你怎么答

| 考题问法 | 标准答案模板 |
|------|------|
| "交流电路中U=U_R+U_L是否成立？" | **不成立！** 交流电路中电压是**相量和**，不是代数和。U=√(U_R²+U_L²) |
| "三表法测阻抗需要哪三块表？" | **电压表、电流表、功率表** |
| "已知U=220V, I=5A, P=880W，求\|Z\|和cosφ" | \|Z\|=220/5=**44Ω**；cosφ=880/(220×5)=**0.8** |

### 三、交流电路 ≠ 直流电路 核心红线 ★★★★★

```
┌─────────────────────────────────────────────────┐
│  ⚠️ 交流电路中千万不能用代数加减！                 │
│                                                 │
│  直流电路：U = U₁ + U₂    （代数加）               │
│  交流电路：U̇ = U̇₁ + U̇₂    （相量加！）              │
│                                                 │
│  考试坑：                                         │
│  "U = U_R + U_L 对吗？"  →  错！                 │
│  "I = I_RL + I_C 对吗？"  →  错！                 │
│  正确：U = √(U_R²+U_L²)，I = √(I_R²+I_C²)        │
│  （仅在纯电阻/纯电感/纯电容时可代数加减）            │
└─────────────────────────────────────────────────┘
```

### 四、真题模拟 3题

**题1**（★★★★★）：用三表法测量某感性负载，测得U=220V，I=2A，P=352W。求：(1)阻抗模\|Z\|；(2)功率因数cosφ；(3)等效电阻R；(4)等效感抗X_L。

> **套模板**：(1) \|Z\|=220/2=**110Ω**；(2) cosφ=352/(220×2)=**0.8**；(3) R=352/4=**88Ω**；(4) X_L=√(110²−88²)=√(12100−7744)=√4356=**66Ω**

**题2**（★★★）：感性负载两端电压U_R=80V，U_L=60V。问总电压U=？

> **标准答案**：**U=√(80²+60²)=100V，不是140V！** 因为电阻电压和电感电压相位差90°，必须用相量加法。

**题3**（★★★★）：画出感性负载的阻抗三角形，并标注R、X_L、\|Z\|、φ。

> **标准答案**：横轴=R，纵轴=X_L，斜边=\|Z\|，φ为\|Z\|与R的夹角。cosφ=R/\|Z\|。

### 二、并联电容补偿 ★★★★★

```
┌─────────────────────────────────────────────────┐
│       并联电容提高功率因数（必考计算题！）           │
│                                                 │
│  方法：在感性负载两端并联适当电容                   │
│  原理：电容提供容性无功 → 抵消感性无功             │
│                                                 │
│  ⚠️ 并联电容后：                                  │
│  · 感性负载自身工作状态不变（端电压不变）            │
│  · 负载自身 cosφ 不变                              │
│  · 总电流减小 → 总功率因数提高                     │
│  · 总视在功率 S 减小                              │
│  · 有功功率 P 不变                                │
│                                                 │
│  口诀：「并电容、容抵感、总流降、因提上来」            │
└─────────────────────────────────────────────────┘
```

### 三、欠补偿 / 过补偿 ★★★★

```
┌─────────────────────────────────────────────────┐
│  并联电容 C 从小到大变化时：                       │
│                                                 │
│  C=0：纯感性，cosφ 最低                           │
│  C↑：容性无功增加，总无功减小，I↓，cosφ↑           │
│  C=最佳值：完全补偿，cosφ=1，I 最小                │
│  C再↑：过补偿，容性无功过量，I 反而上升，cosφ↓      │
│                                                 │
│  判断方法：总电流 I 随 C 增大先减小后增大            │
│           I 最小时 → 完全补偿（cosφ=1）             │
│                                                 │
│  口诀：「C小欠、C大过、I最小处刚刚好」               │
└─────────────────────────────────────────────────┘
```

#### 考试怎么问 → 你怎么答

| 考题问法 | 标准答案模板 |
|------|------|
| "并联电容后感性负载自身cosφ改变吗？" | **不改变**。负载端电压不变，自身参数不变 |

| "已知原来cosφ₁=0.6要提高到cosφ₂=0.9，怎么算电容？" | 套补偿公式（考试一般不要求精确计算电容值，但要知道并电容即可实现） |
